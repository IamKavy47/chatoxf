Introduction

Overview & Vision

ChatOXF is a next-generation messaging platform designed to bridge the gap between static web pages and fully reactive applications.

Core Philosophy:

Fast, Private, and Intelligent.

  • Seamless real-time messaging.
  • Integrated AI Assistant for productivity.
  • Modern, Dark-Mode aesthetics.

Why ChatOXF?

Relevance in Computer Science

Low Latency

Uses WebSocket-based subscription models instead of traditional polling.

Data Sovereignty

Hosted on Indian servers to ensure better speed and data privacy.

AI Integration

Demonstrates practical implementation of LLMs in web apps.

Cross Platform

Available as a PWA and a dedicated Android Application.

Limitations & Challenges

Current Constraints

Scalability Costs

While the free tier supports university projects, scaling to thousands of users requires paid Convex plans (approx. â‚¹1,200/mo).

Media Support

Currently supports text and emojis. Video/Voice calling and heavy file sharing are yet to be implemented.

Native Performance

The Android app relies on WebView technology, which is slightly less performant than Kotlin-native code.

Encryption

Messages are secure in transit (HTTPS), but full End-to-End Encryption (E2EE) is in the R&D phase.
